2 Avec le Type < T >

la Maison est pas est limitée pas réutilisable; il ne peut contenir rien d’autre qu'Une Personne (class Person). Nous ne voulons pas pour écrire un nouveau type de support pour chaque type différent. Pour ce faire, nous utilisons un paramètre de type au lieu de Person. Pour définir un type générique, ajoutez des crochets < > contenant un ou plusieurs espaces réservés génériques et placez cette spécification générique après le nom de la classe. Ici, l’espace réservé générique T représente l’inconnu et est utilisé dans la classe comme s’il s’agissait d’un type régulier :nome de la class < T > la syntaxe est :class exemple < T > le T est juste une illutration vous pouvez mettre n'impore quelle lettre V E out autre à la T 


    class Maison < T>(val valeur: T) {
    
    fun toutType():T=valeur
    
    }
    
 _________________________________

 fun main(){

    val perso= Maison(Person("Korro"))

    println (perso.toutType().name)
