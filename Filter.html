1 La Fonction filter

Cette fonction vous permet de rechercher, ou filter, une collection selon certains critères que vous lui transmettez à l’aide d’un lambda. Pour la plupart des collections, le filtre renvoie une liste qui inclut tous les éléments qui correspondent à vos critères, qui vous pouvez ensuite l’utiliser ailleurs dans votre code. Toutefois, s’il est utilisé avec une carte, il renvoie une carte. Le le code suivant 

 

fun main()  {

    val list= listOf(1,2,3,4,5)


    var f=list.filter{ it > 3}

    println(f)

    
     }

Résultat:

    Filter:[ 4 ,5 ]

Explication: dans cet exemple on avait crée une simple liste de type entier(Int ) et nous voulions savoir les élements qui sont superieus à 3 et pour faire ça ,nous avions utilisé la fonction filter avec une opération de comparaison et la fonction filter nous donne une liste qui correspond à notre demande c est à dire 4,5 sont les seuls élements superieus à 3
