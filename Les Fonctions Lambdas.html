Les Lambdas

Nous allons vous amener dans le monde de lambda et vous allez découvrir comment lambdas sont très puissants et flexibles ,Kolin a une multitude des fonctions qu'on ne serait pas capable de les toutes décrire dans ce tuto. Les lambdas sont des fonctions anonymes sans nom ,les lambdas très utiles surtout quand il s'agit des collections par exemple(trier une liste par nom ou nombre par d'ordre croissant ou décroissant ) L'Expression Lambdas est un type d'objet contenant un block de code. 


    // Simpe exemple de  le Lambda


fun main()  {

    
    //syntaxe de Lambdas
     { x:type -> x }
                 |

   //avec un seul paramètre Int
    var lamb = { x:Int-> x }

      println (lamb (10))

   }

Résultat:

     10


Lambdas avec plusieurs Paramètres

On peut également faire des opérations avec dans les lambdas avec plusieurs paramètres dans ce exemple nous allez découvrir comment faire passer plusieurs paramètres à une fonction Lambda. 


fun main()  {

    var lamb= {x:Int ,y:Int-> x+y }

    println (lamb(5,10))

  }

Résultat:

 
     15

Explication: x représente 5 et y représente 10 donc dans l'expression lambda nous avions fait passer deux paramètres Int(Entier) ces deux paramèttres sont x et y c'est juste un exemple vous pouvez donner n'importe quel nom que vous souhaitez ,moi j'ai juste choisi x et y pour la simplicité et vous pouvez également choisir n'importe quel type {x:Int , y:Int -> x + y } veut dire x+y et retour du résultat.



Affecter un Lambda à une Variable

On peut également affecter un lambda à une variable de la même manière que vous affectez tout autre type d’objet à une variable en définissant la variable à l’aide de val ou var, puis en lui affectant le lambda. Dans cet exemple nous allons faire plusieurs opérations avec de types variés dans des Expressions lamda et affecter ces des lamdas à des variables variées. 

    
fun main() {
        
 var st= {a:Int,b:Int->"a:$a,b:$b"}
 var plus= {a:Int,b:Int-> a+b}
 var moins= {a:Int,b:Int-> a-b}
 var multi= {a:Int,b:Int-> a*b}
 var divis= {a:Int,b:Int-> a/b}

  
//Affecter les Lambdas
  aux variable suivantes 
 var str=st(30,40)
 var plu = plus(20,10)// x+y=(20+10)
 var moin= moins(5,10)//x-y=(5-10)
 var mult= multi(5,10)//x*y=(5*10)
 var divi= divis(50,2)//x/y=(5/10)

  println("a et b =$str")//a,b =(30,40)
  println("x+y =$plu")
  println("x-y =$moin")
  println("x*y =$mult")
  println("x/y=$divi")

}

Résultat:

      a,b =a:30 et b:40
      x+y =30
      x-y =-5
      x*y =50
      x/y =25


Les expressions Lambda ont un type

Comme tout autre type d’objet, un lambda a un type. La différence avec le type d’un lambda, cependant, est qu’il ne spécifie pas de nom de classe que le lambda implémente. Au lieu de cela, il spécifie le type de paramètres et valeur de retour de lambda. 

            
            
    fun main()  {
            
     var st:()->String= {"Bonjour"}

     var tp:(Int,Int)->Int={x:Int,y:Int->x+y}
    
     println(st())
    
     println (tp(10,20))
        
  }

Résultat:

    
    st: Bonjour
    tp: 30
