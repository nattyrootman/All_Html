1 Une classe de Donnée Étudiant

Dans cet exemple on va juste créer une classe de donnéés appelée Étudiant dans la quelle il y'a le nom l'âge l'année et les notes et dans la fonction Main nous avions créer une liste qui accepte une liste type Objet étudiant et pour afficher le nom ,âge année,et notes de chaque Étudiant nous allons faire une boucle(for loop ) qui permet de parcourir toute la collection le but de cet exemple est de montrer comment ultiser les fonctions d'ordre supérieur dans une collection. Dans les prochaines étapes nous utiliserons minByOrNull et maxByOrNull pour trouver la valeur la plus basse ou la plus élevée dans une collection d'objets. 



 
    data class Étudiant(
        val name:String,
        val âge:Int,
        val année:String ,
        val note:Double )

        

fun main()  {

    val list= listOf(
        Étudiant("Amed",25,"4ém",16.9),
        Étudiant("jack",26,"3ém",15.0),
        Étudiant("amel",22,"2ém",15.20),
        Étudiant("moussa",27,"5ém",17.10),
        Étudiant("korro",20,"1ère",14.20))


    println("nom\t    âge\t année \tnote\t")

    for (i in list){

        println("${i.name}    ${i.âge}    ${i.année}     ${i.note}")
    }


      
     }

Résultat:

  nom	  âge  année    note
  Amed    25    4ém     16.9
  jack    26    3ém     15.0
  amel    22    2ém     15.2

  korro   20    1ère    14.2

Explication :Nous avions crée une liste type Étudiant (objet) dans la fonction principale à partir de la classe de donnée Étudiant ou on a listé toutes les données d'Étudiants dans un tableau grace a une boucle qui a traversé toute la liste pour afficher toutes les données de chaque Étudiant Dans les prochains tutos nous nous voudrons savoir la plus petite note et la plus élevée mais aussi le jeune et le plus âgé parmi les Étudiants dans cette base de données



2 Les Fonctions minByOrNull() et maxByOrNull()

pour determiner ou trouver une valeur minimale ou maximale dans une collection plus complexe il faudra utiliser la fonction minByOrNull pour trouver la plus valeur la plus basse et fonction maxByOrNull pour la valeur la plus grande dans une collecttion Dans ce cas nous allons utiliser ces deux fonctions pour trouver l'Étudiant qui a la plus grande note et celui de la pluts petite note ,mais aussi le plus jeune Étudiant et le plus âgé à partir de notre base de donnée Étudiant



3 La Fonction minByOrNull(minimale)

Comment utiliser la fonction minByOrNull pour trouver l'Étudiant qui a la plus petite note mais aussi le plus jeune Edutiant 

     
            
    
    fun main()  {


        val list= listOf(
            Étudiant("Amed",25,"4ém",16.9),
            Étudiant("jack",26,"3ém",17.0),
            Étudiant("amel",22,"2ém",15.20),
            Étudiant("moussa",27,"5ém",17.10),
            Étudiant("korro",20,"1ère",14.20))

            
    println(list.minByOrNull { it.note })

    println(list.minByOrNull { it.âge })



    }

       
    
    jack a la plus petite note qui est 15.0
    et Korro est la plus jeune Étudiante

Résultat:

       jack: note=15.0 
       Korro: âge=20

Explication: dans cet exemple nous avions une liste d étudiants et avions voulu connaitre l'Étudiant qui a la plus petite note qui est Jack et le plus jeune Étudiant qui est Korro ,



5 La Fonction maxByOrNull(maximale)

Comment utiliser la fonction maxByOrNull() pour trouver l'Étudiant qui a la plus petite note mais aussi le plus jeune Edutiant 

     
        data class Étudiant(
            val name:String,
            val âge:Int,
            val année:String ,
            val note:Double )
    
            
    
    fun main()  {
    
        val list= listOf(

            Étudiant("amel",22,"2ém",15.20),
            Étudiant("moussa",27,"5ém",17.10),

         }

Résultat:

            moussa: note=17.10

Explication: dans cet exemple nous avions une liste d étudiants et avions voulu connaitre l'Étudiant qui a la plus grande note qui est moussa et le plus le agé parmi les Étudiant qui est aussi moussa
