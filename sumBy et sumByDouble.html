Les Fonctions sumBy et sumByDouble

Les fonctions sumBy et sumByDouble renvoient une somme des éléments d’une collection. selon certains critères que vous lui transmettez via un lambda. Vous pouvez utiliser ces fonctions pour, par exemple, ajouter ensemble les valeurs de quantité pour chaque élément d’une liste ou renvoyer la somme de chaque élément prix multiplié par la quantité.



1 Fonction sumBy

la fonction sumBy est utilsé pour le type entier par exemple vous avez une liste et vous voulez avoir la somme de tous les élements additionné de cette liste 

 

fun main()  {

    val l= listOf(1,2,3,4,5)

    print("la somme:")
    println(l.sumBy { it })

    
     }

Résultat:

    la somme:15

Explication: dans cet exemple vous avions une liste contenant 5 éléments de type Int (entier) et nous avons une utilisé la fonction sumBy pour donner la somme totale de tous les éléments additionnés cela donne comme cacul :1+2+3+4+5=15 vous voyez comment avec la fonction sumBy() a simplié le cacul ,avec sumBy on a pas besoin de faire tout ce long cacul 1+2+3+4+5 au lieu list.sumBy(it)



2 Fonction sumByDouble()

La fonction sumByDouble() fonctionne de la meme manière que la fonction sumBy la seule difference est que sumByDouble() est flottant (nombre fractionnel) ne fonctionne qu'avec le type Double (flottant) 

 

fun main()  {


    val l= listOf(2.5,4.2,6.9)

    print("la somme:")
    println(l.sumByDouble { it })
 
     }

Résultat:

   
    la somme:13.600000000000001

Explication: dans cet exemple vous avions une liste contenant 5 éléments de type Double (flottant) et nous avons une utilisé la fonction sumBy pour donner la somme totale de tous les éléments additionnés cela donne comme cacul 2.5+4.2+6.9 =13.600000000000001
