1 La Fonction map{}

La fonction map prend les éléments d’une collection et les transforme selon une formule que vous spécifiez. Il renvoie les résultats de cette transformation sous la forme d’une nouvelle liste. 

 

fun main()  {

    val list= listOf(1,2,3,4,5)


    var f=list.map{ it *2}

    println(f)

    
     }

Résultat:

[2, 4, 6, 8, 10]

Explication: dans cet exemple on a crée une liste avec type entier et avions multiplié chacque élément par it*2 it représente chaque élément de la liste dont chaque élément est multiplié par 2
